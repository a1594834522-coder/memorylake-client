## Developer Setup

**Prerequisites:** Python 3.9

**Install:**

```bash
pip install -e .[dev]
```

**Development Commands:**

```bash
./cicd/format.sh                 # Format code
./cicd/check-all-locally.sh      # Run all checks  
./cicd/test-all.sh               # Run tests
```
